# 🌉 다리 건너기

## 개요

- 콘솔창에서 다리 건너기을 할 수 있다.
- 테스트 패키지 설치를 위해 `Node.js` 버전 `14` 이상이 필요하다.
- 다음 명령어를 입력해 패키지를 설치한다.

```bash
npm install
```

- 설치가 완료되었다면, 다음 명령어를 입력해 게임을 실행한다.

```bash
cd src
node App.js
```

### 기능

- 사이즈를 입력받아 랜덤한 다리를 생성하는 기능
- 사이즈의 제한 사항은 3이상 20이하
